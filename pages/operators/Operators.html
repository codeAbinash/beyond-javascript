Operators

JavaScript has many types of operators. JavaScript has unary, binary operators and one ternary operator. Unary operators require one operand, binary require two and ternary requires three. JavaScript has following types of operator -

  • Assignment Operator
  • Arithmetic Operator
  • Comparison Operator
  • Logical Operator
  • Type Operator
  • String Operator
  • Conditional Operator
  • Comma Operator
  • Unary Operator
  • Relational Operator
  • Bitwise Operator

Unary Operator

Unary Operator is an operator that acts upon a single operand in an expression or a statement. Some examples of unary operator in javascript are typeof, delete, void, ++, -- etc.

typeof operator

typeof operator returns the data type of the operand (in string data type).

delete Operator

The delete operator can delete object property. on successful deletion, it returns true, else returns false .
