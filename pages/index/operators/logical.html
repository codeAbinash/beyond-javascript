Logical Operator

Generally Logical Operators are used with Boolean(true or false) values. They can also be used with non boolean values in JavaScript. There are three main logical operators in JavaScript.

  • Logical AND
  • Logical OR
  • Logical NOT

Before you start learning logical operators, you need to learn about truthy and falsy values

Falsy

A falsy value is consider false in Boolean context.

false, null, NaN, undefined, 0, Empty String, Objects Having The [[IsHTMLDDA]] Internal Slot are considered false. All other values are considered true.

Truthy

All values are truthy expect those mentioned in falsy section.

Logical AND (&&)

The Logical AND Operator will return true if and only if all operands are true, else it will return false. Generally it returns the first falsy operand when evaluating left to right or the value of the last operand if all of the operands are truthy.

Non boolean values, returns the first falsy operand. If all operands can be evaluated(left to right) to true, returns the last operand.

Short-Circuit Evaluation

As the AND operator returns the first falsy operand and stops evaluating. It can be used as a Short-Circuit Evaluation. Look at the following example.
