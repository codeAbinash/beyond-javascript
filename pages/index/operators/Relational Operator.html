Relational Operator

In JavaScript Relational Operators return a Boolean value ( true or false ). There are two relational operator in JavaScript.

  • in operator
  • instanceof operator

in Operator

The in operator returns true if the first operand is in the second operand (object or it's prototype chain).

instanceof Operator

The instanceof operator returns true if the object type (constructor) of the first operand appears in the prototype chain of the second operand.

Simply, it returns true if the type of the first operand is of the type of the second operand.
