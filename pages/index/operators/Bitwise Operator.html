Bitwise Operator

In JavaScript Bitwise Operations are performed on 32 bit binary number system. Before bitwise operation is performed, JavaScript converts operands to 32 bit number system. For example -

2 can be represented as 00000000000000000000000000000010 .

5 can be represented as 00000000000000000000000000000101 .

There are seven bitwise operators in JavaScript.

  • Bitwise AND
  • Bitwise OR
  • Bitwise XOR
  • Bitwise NOT
  • Left Shift
  • Right Shift
  • Zero Fill Right Shift

Bitwise AND (&)

The Bitwise AND operator first converts the operands to 32 bit binary number system and returns 1 in each bit position where both of the operand's bits are 1 else returns 0.

Bitwise OR (|)

The Bitwise OR operator returns 1 in each bit position where there is at least one 1 in that bit position of any operand. else returns 0.

Bitwise XOR (^)

The Bitwise XOR operator returns 1 in each bit position where operand's bits are not same in that position.

Bitwise NOT (~)

The Bitwise NOT operator is a unary operator. It inverts the bit of the operand.
