Unary Operator

Unary Operator is an operator that acts upon a single operand in an expression or a statement. Some examples of unary operator in javascript are typeof, delete, void, ++, -- etc.

  • typeof Operator
  • delete Operator

typeof operator

typeof operator returns the data type of the operand (in String data type).

Maybe you have noticed typeof null is object. It is a bug in JavaScript. It should return null. It's never been fixed and will not be fixed in future because there are too much code relies on this bug on the web.


delete Operator

Delete object property

The delete operator can delete object property. on successful deletion, it returns true, else returns false .

{ name: 'Abinash' }

Returns true if the property does not exist in the object. In the following example, property z is deleted from object data. It returns true.

true
{ x: 10, y: 20 }

Any variable declared using var, let or const cannot be deleted using delete.

Delete array elements
