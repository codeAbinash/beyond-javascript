Logical Operator

Generally Logical Operators are used with Boolean(true or false) values. They can also be used with non boolean values in JavaScript. There are three main logical operators in JavaScript.

  • Logical AND
  • Logical OR
  • Logical NOT

Other Logical Operators -

  • Logical Nullish Operator

Before you start learning logical operators, you need to learn about truthy and falsy values.

Falsy

A falsy value is consider false in Boolean context.

false, null, NaN, undefined, 0, Empty String, Objects Having The [[IsHTMLDDA]] Internal Slot are considered false. All other values are considered true.

Truthy

All values are truthy expect those mentioned in falsy section.


Logical AND ( && )

The Logical AND Operator will return true if and only if both operands are true, else it will return false.

Non Boolean Operands

For non boolean operands the logical AND operator returns the first falsy (left to right evaluation) operand, or the last operand if both of the operands are truthy.

Short-Circuit Evaluation

As the AND operator returns the first falsy operand and stops evaluating. It can be used as a Short-Circuit Evaluation. Look at the following examples.

Logical OR ( || )

The Logical OR operator returns true if at least one of the operand is true. If both operands are false , it returns false.

Non Boolean Operands

For non boolean operands the logical OR operator returns the fist truthy operand (left ot right evaluation). If both operands are falsy returns the last operand.

Short-Circuit Evaluation

As the OR operator returns the first truthy operand and stops evaluating. It can be used as a Short-Circuit Evaluation. Look at the following examples.

In the above example the first operand is 500 (truthy value), so the OR operator stops evaluating and returns that value.

Logical NOT ( ! )

The Logical NOT operator is a Unary operator (works on single operand). It takes truthy value and returns false, takes falsy value and returns true.

Double NOT

Using the NOT operator twice returns true if the operand is truthy and returns false if the operand is falsy.

It works like Boolean() global function which you will read later.

Logical Nullish Operator ??

The Logical Nullish operator returns the second operand if the first operand is null or undefined. Else it will return the left operand.

You can imagine this operator like the || operator. The difference is, logical OR operator checks if the first operand is falsy, returns the second operand. The logical Nullish operator checks if the first operand is null or undefined, returns the second operand. It is a special case of logical OR operator.
