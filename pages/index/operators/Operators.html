JavaScript Operators

Operators

An operator is a special type of symbols(single character or multi character) used to perform operation on operands.

JavaScript has many types of operators. JavaScript has unary, binary operators and one ternary operator. Unary operators require one operand, binary require two and ternary requires three. JavaScript has following types of operator -

  • Assignment Operator
  • Arithmetic Operator
  • Comparison Operator
  • Logical Operator
  • Type Operator
  • String Operator
  • Conditional Operator
  • Comma Operator
  • Unary Operator
  • Relational Operator
  • Bitwise Operator
