Increment ++

Increment ++ operator is a unary operator. It increments the value of the operand by 1(adds 1 with the operand and stores the result in it). There are two ways to use this operator - Prefix and Postfix.

Prefix

First it increments the operand by one and returns the value of it.

6
6

In the above example the Prefix ++ operator increments the value of x(adds 1 to it). So now x becomes 6. And then it returns the value(6). The value(6) is now assigned to y.

Postfix

First it returns the value of the operand and then increments the value of it.

101
100

In the above example first the value of x is returned and assigned to y. Then the value of x is incremented.
