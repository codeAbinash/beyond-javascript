Switch - Case

The switch statement is used to evaluate an expression and execute a specific code block depending on the result of evaluated value.

At first the switch statement evaluates expression and matches the evaluated value with each value given after case. If the value is matched, the code will start executing after that case value : until it gets a break keyword. If no match found then the code written after default : will be executed.
