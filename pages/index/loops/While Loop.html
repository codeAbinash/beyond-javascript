While Loop

The while statement executes a block of code repeatedly while the given condition evaluates to true.

First it checks for the specified condition. If the condition evaluates to true the code block or statement written after while (...) will start executing. When the execution is done, again it checks the condition, if it evaluates to true, again the statement or the code block will be executed. As soon as the condition evaluates to false, the loop stops executing and breaks from it.

The break Statement

break statement is used to terminate a loop, switch or a labeled block. If break is used without a label, it will terminate the innermost loop. If used with a label, it will terminate the specified label block or statement.

Example : Without a label name

Example : With a label name

The continue statement

The continue statement can be used to stop executing the current iteration and start the loop again. It can be used with o r without label like the break statement.

Example : Without a label name

1 2 4 5

Example : With a label name
