Do While Loop

Do while loop repeats until a condition is evaluated to false.

Do while loop always executes the code block or statement at least once before the condition is checked. And then if the condition evaluated to true, it executes the block or statement again and so on. If the condition becomes false, it will stop executing the loop and the control will exit the loop.
