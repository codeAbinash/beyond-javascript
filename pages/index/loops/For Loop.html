For Loop

The for loop executes a code block or statement repeatedly until the provided condition is evaluated to false.

initializer is used to initialize one or more variable before the loop starts. Generally it is used to initialize counter variables, for example let i = 0. initializer is executed once. Initialized variable(s) are locally scoped inside the loop (for let & const only, var is global or function scoped). It is optional, you can skip it.

condition is executed in each iteration of the loop and before the loop starts and . If the condition evaluated to false, the loop stops executing and the control exits from the loop. This condition is optional. You can omit it. If you do so, the loop will never terminate unless terminated with break statement inside of the loop.

last_expression is executed after each iteration of the loop. Generally it is used to increment or decrement initialized values.
