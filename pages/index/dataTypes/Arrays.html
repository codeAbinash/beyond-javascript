JavaScript Arrays

Arrays

Array is a single variable that is used to store different elements in a list.

Values are written inside [] brackets and separated using commas(,). Check the following example.

[ 1, 4, 'hi', -88, false ]
5

Unlike C/C++, you can store different types of data in an array. Check the above example, there are different types of data are stores in the array values.

Use length property to get the length of the array.

Access Array Elements

To access an element of an array use [index] followed by the name of the array. Array index starts with 0, means you can access the first element of the array using arrName[0]. The second element using arrName[1], the third element using arrName[2] and so on.

1
4
[ 'Abinash', 5, 'hi', -88, false ]

You can Define an empty array and insert values into it later.

[]
[ 'Anju', 'Abinash', 'Suman', 'Rohan' ]

Be careful when you are using empty array initially. If you skip any index there will be created empty values (undefined) in that index of the array.

undefined
[ 'Me', <1 empty item>, 'You' ]

If you access index which is out of the range of the array, it will return undefined

undefined
