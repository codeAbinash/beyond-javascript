Data Type null

null represents nonexistence or nothing. In JavaScript null is an object. It represents absence of any object value. Every object is derived from null value.

typeof null returns "object". It is a bug in JavaScript. It should return "null". It's never been fixed and will not be fixed in future because there are too much code relies on this bug on the web.
