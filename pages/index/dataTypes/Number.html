Number()

The Number() function tries to convert any value to Number type. If it fails, it will return NaN (not a number).

NaN

NaN means Not a Number. When JavaScript tries to convert any other data type to number, if it fails, it returns NaN. typeof NaN is number. You must be familiar with NaN if you read previous chapters.

Number Methods

valueOf()

Returns the value (as number primitive).

isNan()

Returns true if the number is NaN else false.

isInteger()

Returns true if the number is integer else false.

isSafeInteger()

Returns true if the number is in the range of safe integer else false.

isFinite()

Returns true if the number is finite else false

parseInt()

Parse integer from a value and returns it, if it fails, it will return NaN

Do a lot of experiment yourself, it will be better to understand clearly.

parseFloat()

Parse floating point number from a value and returns it, if it fails, it will return NaN

toPrecision()

Returns a string.

toExponential()

toString()
