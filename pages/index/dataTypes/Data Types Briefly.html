Data Types in JavaScript, Briefly

This is basics of JavaScript data types. We will discuss about each data type in detail later. You can skip this page if you are not beginner.

JavaScript has eight data types -

  • null
  • undefined
  • Boolean
  • Number
  • BigInt
  • String
  • Symbol
  • Object

Object is non-primitive data type. All other 7 data types are primitive.

JavaScript is a loosely typed language. This defines the language does not require a variable to be defined using a specific data type. It is a dynamically typed language. Variables can be assigned to any data type. Data types are automatically converted as needed while execution.
