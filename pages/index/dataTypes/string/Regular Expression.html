Regular Expression

A Regular Expression is a pattern of characters. Regular Expressions are used to match character combination in a string.

Regular Expressions are objects in Javascript

First you have to learn Regular Expression then you can use it in Javascript.

regexlearn.com , regexr.com, regex101.com are very good website for learning and practicing regular expressions.

Create Regular Expression

If the regular expressions is dynamic, it is better to create a regular expression using constructor.

Regex Properties

source

Returns the pattern of the regular expression

global

Checks if global modifier is set or not.

ignoreCase

Checks if ignore case modifier is set or not.

multiline

Checks if multiline modifier is set or not.

lastIndex

Returns the index from where to start the next match.

String Methods

There are some string methods where you can use regular expressions. search(), match() replace() split()

match()

Matches a regular expression and returns an array. If not match found returns null.

[ 'I', 'G', 'B' ]
null

matchAll()

Returns an array of all results after matching a string

search()

Matches a string for a regular expression and returns the first match if not found returns -1.

Found at index : 17

replace()

Returns a new string after replacing.

nine one two three one
nine nine two three nine

You can use nth group using $n.

Pass function as argument for replacing.

replaceAll()

Replaces all matching patterns.

split()

Divides the string into substrings and returns arrays for the matching pattern.

[ 'Hello', 'Pekachu' ]
[ 'Abinash', 'Ananta', 'Tukai', 'Suman' ]
