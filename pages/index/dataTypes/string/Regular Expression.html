Regular Expression

A Regular Expression is a pattern of characters. Regular Expressions are used to match character combination in a string.

Regular Expressions are objects in Javascript

First you have to learn Regular Expression then you can use it in Javascript.

regexlearn.com , regexr.com, regex101.com are very good website for learning and practicing regular expressions.

Create Regular Expression

If the regular expressions is dynamic, it is better to create a regular expression using constructor.

Regex Properties

source

Returns the pattern of the regular expression

global

Checks if global modifier is set or not.

ignoreCase

Checks if ignore case modifier is set or not.

multiline

Checks if multiline modifier is set or not.

lastIndex

Returns the index from where to start the next match.

If you don't understand this example, read test() method first.
