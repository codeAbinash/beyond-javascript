Template Literals

Template Literals are determined using ` (backtick characters).

It allows multi-line strings.

This is a Multi Line String

You can use single or double quote directly inside a string.

You can use 'single quote' or "double quote" directly

String Interpolation

You can embed placeholders using ${placeholder}.

Hello my name is Abinash and I am 20 years old.

Read the above example carefully. ${name} is replaced by the value of the variable name and ${age} is replaced by age variable. Read more examples to make it clear.
