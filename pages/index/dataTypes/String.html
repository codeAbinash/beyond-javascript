String

Before you know strings, you have to know what a character is.

Character

A character is a unit of information that roughly corresponds to a grapheme, grapheme-like unit, or symbol, such as in an alphabet or syllabary in the written form of a natural language. Read more on Wikipedia.

See UTF - 16 characters for better understanding. They all are characters.

String

A string is a sequence of characters used to represent a text.

String is primitive data type in Javascript. To use a string literals in Javascript write zero or more characters inside double quote or single quote or you can use backticks(`).
