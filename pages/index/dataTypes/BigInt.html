Data Type BigInt

BigInt

BigInt is a primitive data type in Javascript. It can represent integers with arbitrary precision. So you can store very large integer value in a BigInt type variable.

To create a primitive BigInt variable, write a number and then append a n at the end of it.

155n
bigint

Operations with Number primitives is not possible.

TypeError: Cannot mix BigInt and other types, use explicit conversions

BigInt cannot contain decimal numbers.

BigInt()

The BigInt() function converts value to primitive Bigint type.

You can store larger values than Number.MAX_SAFE_INTEGER.
