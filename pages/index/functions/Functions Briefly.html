Functions

Parameters & Arguments

Parameters are the values which are defined in the function inside ().

Arguments are the values that are passed inside the function during calling of the function.

Return Vales

To return a value from a function, use return keyword. When return is used in a function, it immediately stops executing any code inside the function and returns a value where the function was called.
